## 新しく知ったこと
```python
l1 = [1, 2, 3, 4, 5]
l2 = [7, 6, 5, 4]
result = set(l1) | set(l2)    # 和集合
print(f'すべての要素 : {result}')    # すべての要素 : {1, 2, 3, 4, 5, 6, 7}

l1 = [1, 2, 3, 4, 5]
l2 = [7, 6, 5, 4]
result = set(l1) - set(l2)    # 差集合
print(f'l1のみに格納されている要素 : {result}')

l1 = [1, 2, 3, 4, 5]
l2 = [7, 6, 5, 4]
result = set(l1) ^ set(l2)    # 対称差集合　「^」ハット
print(f'どちらか一方に格納されている要素 : {result}')    # どちらか一方に格納されている要素 : {1, 2, 3, 6, 7}

# 部分集合
l1 = [1, 2, 3, 4, 5]
l2 = [3, 4, 5]
print(f'l1はl2に含まれている : {set(l1).issubset(set(l2))}')    # l1はl2に含まれている : False
print(f'l2はl1に含まれている : {set(l2).issubset(set(l1))}')    # l2はl1に含まれている : True
```
