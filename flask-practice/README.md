# ✅ チェックリスト（メモアプリ）

## 0. 事前条件

- [ ] DB初期化済み（ユーザー登録できる状態）
- [ ] ログインしていない状態から開始（別ブラウザ/シークレット推奨）

---

## 1) 手動テスト（正常系）

### サインアップ（ユーザー登録）

- [ ] `/signup` を開く  
  **期待**：新規ユーザー登録画面が表示される

- [ ] userid/password を入力して登録できる  
  **期待**：登録成功メッセージが出て `/login` に遷移する

---

### ログイン

- [ ] `/login` を開く  
  **期待**：ログイン画面が表示される

- [ ] 正しい userid/password でログインできる  
  **期待**：「ログインしました。」が出て `/`（トップ）へ遷移する

---

### メモ作成

- [ ] `/regist` を開く  
  **期待**：新規登録フォームが表示される

- [ ] タイトル/本文を入力して作成できる  
  **期待**：作成後 `/` に戻り、一覧に表示される

---

### メモ編集

- [ ] `/<id>/edit` を開く  
  **期待**：編集フォームが表示される

- [ ] 編集して更新できる  
  **期待**：更新後 `/` に戻り、一覧に反映される

---

### メモ削除

- [ ] `/<id>/delete` を開く  
  **期待**：削除確認画面が表示される

- [ ] 削除できる  
  **期待**：削除後 `/` に戻り、一覧から消える

---

### ログアウト

- [ ] ログアウトできる（POST）  
  **期待**：「ログアウトしました。」が出てログイン画面へ
